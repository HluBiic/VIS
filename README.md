* mvn clean package
* ConsoleApp.java -> run as Java Application (Eclipse IDE)
* WebApp.java -> run on Tomcat v10.1 server

* Console app is used by the Casters for basic CRUD operations with matches.
* Web app is used by players to view and export the data about tournaments and matches.
* Implemented Use Cases: 
	* UC03: CRUD operations for match
	* UC10: Detailed report of tournament
	* UC14: Export of the detailed report
